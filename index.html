What is Phishing?

Phishing is a cybercrime where attackers impersonate legitimate organizations to steal sensitive information like usernames, passwords, and credit card details.

üí° Key Fact: AI-generated phishing emails have a 54% click rate compared to 12% for human-written ones.

Types of Phishing Attacks

üìß Email Phishing

üéØ Spear Phishing

Module 2: Recognizing Phishing Emails

Module 2 of 5

Warning Signs Checklist

üö® Sender Issues

  • Generic greetings (Dear Customer)
  • Suspicious email addresses
  • Public domain emails from companies
  • Spoofed sender names

üìù Content Red Flags

  • Urgent action required
  • Threats of account closure
  • Too good to be true offers
  • Spelling and grammar errors
  • Unexpected attachments

üîß Technical Indicators

  • Mismatched URLs when hovering
  • Suspicious links
  • Requests for login credentials
  • Unusual file extensions
  • Missing security indicators

Urgency

üéÅ

Reciprocity
